This is a list with the set of library files and executable you need to do Cryptography, e.g. encryption, decryption, using OpenSSL.

## Windows Machine
1) libcrypto, e.g. libcrypto-1_1-x64.dll
2) libssl, e.g. libssl-1_1-x64.dll
3) openssl.exe
4) zlib1.dll
